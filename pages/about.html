About Card Ref Pro

Card Ref Pro started as a simple idea: give Trading Card Game (TCG) players a single, ultra-fast page to look up cards and rules without hunting through forums or slow databases. Over time it will grow into a full-featured toolbox for players, collectors, judges, and content creators.

Why this site exists

When a match or deck build depends on a single line of card text, seconds matter. Card Ref Pro prioritizes speed and clarity — compact card views, readable rulings, and keyboard-friendly search make it easy to find what you need in a hurry.

Core and Planned Features

The platform already supports fast card lookups and will include the following features as it grows:

  • Card search — instant lookup by name, partial text, or card ID.
  • Card filtering — filter by set, rarity, type, cost, attack/defense and more.
  • Table view — customizable, sortable columns and column visibility to analyze lists of cards.
  • Deck building — create, edit, and test decks with an easy-to-use deck editor.
  • Deck exporting — export lists to common formats (CSV, plaintext) and shareable links.
  • TCGplayer integration — send deck lists for bulk purchasing to TCGplayer.
  • Random hand & draw generator — simulate shuffles, draws, and opening hands.
  • More utilities — card comparison, banlist alerts, price lookups, and community requests.

Current Coverage

  • Yu-Gi-Oh! — primary dataset and current focus.

Roadmap

Next up: expand coverage to Pokémon TCG, Magic: The Gathering, One Piece, and other major games. Feature priorities will be guided by community feedback; early goals include a robust card filtering, table analytics, and deck building.

Get Involved

This project is built for players. If you have feature requests, data contributions, or want to report an issue, please open an issue or contact the project maintainers. Contributions and feedback will help determine which games and features come next.
